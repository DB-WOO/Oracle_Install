● 2. DB 수동 생성을 위한 파라미터 파일 생성 ( pfile ) <br>
<br>
<br>
데이터 파일이 들어갈 디렉토리 생성<br>
감사(Audit) 디렉토리 생성<br>
<br>
[oracle@ora19c ~]$ mkdir -p /u01/app/oracle/oradata/abc<br>
[oracle@ora19c ~]$ mkdir -p /u01/app/oracle/admin/abc/adump<br>
[oracle@ora19c ~]$<br>
[oracle@ora19c ~]$ cd /u01/app/oracle/oradata<br>
[oracle@ora19c oradata]$ ls<br>
ORCL  abc<br>
[oracle@ora19c oradata]$<br>
[oracle@ora19c oradata]$ cd /u01/app/oracle/admin/<br>
[oracle@ora19c admin]$ ls<br>
abc  orcl<br>
[oracle@ora19c admin]$ cd abc<br>
[oracle@ora19c abc]$ ls<br>
adump<br>
[oracle@ora19c abc]$<br>
<br>
파라미터 파일에는 적었는데, 실제 해당 디렉토리가 만들어지지 않았다면 startup의 nomount 단계에서 에러가 난다.<br>
<br>
<br>
<br>
<br>
$ORACLE_HOME/dbs 경로에 pfile 생성 ( orcl[SID].ora ) <br>
<br>
※ [SID]를 8글자 이하로 한다. (orcl_manual 이라는 11글자로 했다가 에러나서 처음부터 다시 했다.)<br>
